php–变量

1、一般情况下一个变量只能装一个（不要太贪心）值。

2、当我们放进去一个苹果后，再放进榴莲的时候，苹果就被替换掉，再放 入香蕉，那么榴莲就被替换了。

如何定义变量

概念：定义变量就是向服务器的内存（服务器的内存，我们可以当做一个大仓库）申请空间（我们可以理解为一个子仓库），用来存储数据。

1、变量名前面 必须 用 $ 来标识。

变量名的命名规则

1、变量名必须以字母或下划线 “_”开头，不可以用 数字 开头。

2、变量名不允许包含空格，当变量名由多个单词组成，建议使用“_”，或者用驼峰命名法。

3、在PHP中变量名是区分大小写的。

数据类型

string ：字符串类型;

当字符串很长的时候怎么办？

boolean : 只有2个值，TRUE FALSE，不区分大小写。echo”指令输出布尔 类型时，如果是“true”则输出的是“1”，“false”则什么也不输出。

整型：

类似于常见的整数。它可以用十进制、八进制、十六进制指定。

十进制就是日常使用的数字；

八进制，数字前必须加上“0”(这个0是阿拉伯数字0，可不是英文字)；

十六进制，数字前必须加“0x” (这个0也是阿拉伯数字0)。

浮点型

浮点型（浮点数、双精度数或实数），也就是通常说的小数，可以用小数点或者科学计数法表示。科学计数法可以使用小写的e，也可以使用大写的E。

什么是继承

1.继承就是在原有的对象上，略作修改，得到一个新的对象。

2.不影响原有对象的功.

使用call 方法：

call 方法是 用来继承属性的。

继承方法就不能用call 了，要用 for in。

cal方法是为了改变this的指向。
